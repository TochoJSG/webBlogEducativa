Algebra Lineal

Matematicas usadas para manipular arreglos o matrices(tema con relacion con las bases de datos y la programacion como tal), vectores (usados en fisica e ingenieria en general), espacios vectoriales (usado en fisica, una de las aplicaciones es para comprobar la existencia de multiversos y cosas asi) y sistemas de ecuaciones lineales (tema vinculado con matrices, relacionando coeficientes de diferentes variables)

Matriz

Arreglo bidemensional de numeros. Conjuntos de numeros organizados en formato de tabla, con filas y colunmas.
Una Matriz se representa por una letra Mayuscula, y cada uno de sus elementos se representa por la misma letra pero para simbolizar que pertenece a la matriz en en minuscula.
Cada elemento dentro del arreglo o conjunto de numeros representa un coeficiente de un Sistema de Ecuaciones o transiciones lineales dada una base.

Nota: si el numero de las dilas es igual al numero de columnas existentes se pueden sumar o restar las matrices.

Suma de A & B tal que Cij = aij + bij

Si A = [aij] && K=Real Se define o obtiene otra Matriz KA = C; K*A[aij] = C[cij]

Propiedades de Determinantes

  1. Si A = I ===> |A| = | I | = 1
  2. Si Renglon o Columna de Aesta Compuesta de puros 0's ===> |A| = 0
  3. Si A |A| = 0 ===> A es singular inversamente
  4. Si A es singular ===> |A| = 0
  5. Si A,B son n*n ===> det( A, B ) = det( A ) * ( det B )
    Consecuencia:
    Si B = A -1 => det ( AA -1 ) = ( det A ) * ( det A -1 )
    det I = 1 = ( det A ) * ( det A -1 )
    det A -1 = 1 / det A ; det A = 1 / det A -1
  6. Si A = B ===> det B = det A = det A
  7. Si B = KA ===> det B = K n det A
  8. Si B se forma intercambiando 2 renglones de A ===> det B = - det A
  9. Si B se forma multiplicando un renglon de A por un numero r ===> det B = r det A
  10. Si B se forma sumando a un Renglon el multiplo de otro renglon de A ===> det B = det A
